# react-shopping

#### 项目介绍

移动Web APP开发之实战美团外卖

#### 软件架构

React全家桶，移动Web开发技术全解析，理解 Hybrid 混合模式开发

#### 安装教程 

1.安装依赖
   - npm install 

2.dev环境服务启动
   - npm run dev

3.build环境服务启动：
   - npm run build

*启动build之后，项目打包过后的文件会自动复制到waimaiServer的public文件下*

#### 使用说明
* 简介：iOS webview源码，用来承载项目页面的webview壳子
* 运行：安装xcode9.4.1 以上，双击 *WebviewDemo.xcodeproj* 即可运行
* waimai_webapp: 项目页面逻辑源码
